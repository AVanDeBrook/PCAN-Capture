PCAN-Basic Documentation
PCANBasic Module

The Peak.Can.Basic module contain types and classes for using the PCAN-Basic API within Python 2.6 programming environment, in order to handle with PCAN devices from PEAK-System.

Classes
 
Class 
Description 
 
PCANBasic 
Defines a class which represents the PCAN-Basic API. 
Structures
 
Structure 
Description 
 
TPCANMsg 
Defines a CAN message. The members of this structure are sequentially byte aligned. 
 
TPCANTimestamp 
Defines a time-stamp of a CAN message. 
 
TPCANMsgFD 
Defines a CAN message with flexible data rate.
