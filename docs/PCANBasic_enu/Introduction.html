PCAN-Basic Documentation
Introduction

Welcome to the documentation to PCAN-Basic. 

 

The PCAN system of the company PEAK-System Technik GmbH consists of a collection of Windows Device Drivers, which allow the real-time connection of Windows applications to all CAN busses physically connected to a PC. 

 

PCAN-Basic, successor of PCAN-Light, is a simple programming interface to the PCAN system. Via the PCAN-Basic Dll it is possible to connect own applications to the Device drivers and the PCAN hardware, to communicate with the CAN busses.
