PCAN-Basic Documentation
Types

The PCAN-Basic API defines the following types: 

 

Name 
Description 
TPCANHandle 
Represents a PCAN hardware channel handle. 
TPCANStatus 
Represents a PCAN status/error code. 
TPCANDevice 
Represents a PCAN device. 
TPCANParameter 
Represents a PCAN parameter to be read or set. 
TPCANMessageType 
Represents the type of a PCAN message. 
TPCANType 
Represents the type of PCAN hardware to be initialized. 
TPCANMode 
Represents a PCAN filter mode. 
TPCANBaudrate 
Represents a PCAN bit rate register value. 
TPCANBitrateFD 
Represents a bit rate string with flexible data rate. 
TPCANTimestampFD 
Represents the timestamp of a CAN message with flexible data rate.
